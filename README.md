# codeigniter-export-dynamic-chunking
```bash
I made an small project export excel on codeigniter 3 (it could be work on codeigniter 4 as well) with library phpspreadsheet and using chunk method.

# requirement
```bash
- PHP v8^
- codeigniter 4
- composer
- database (i'm using PostgreSQL)

# installation
First init project is required do 
```bash
$ composer install
```
or
```bash
$ composer update 
```
and then do
```bash
$ composer dump-autoload
```
